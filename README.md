# Azure App Configuration Actions

This repository contains the following actions for managing Azure App Configuration:

## Actions Overview
| Action             | Name                            | Description                                    |
|--------------------|---------------------------------|------------------------------------------------|
| create-snapshot    | Azure App Configuration Create Snapshot | Creates a snapshot in Azure App Configuration using Azure CLI. |
| set-keyvalue       | Azure App Configuration Set Key-Value | Sets a single key-value pair in Azure App Configuration using Azure CLI. |
| sync-config        | Azure App Configuration Sync    | Azure App Configuration Sync of Properies and Feature Flags using Azure CLI. |
